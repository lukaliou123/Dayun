# 逃离大运 🚗💨 | Escape from DAYUN

一款基于 Three.js 的 3D 伪透视赛车游戏。驾驶红色小车，全力逃离身后疯狂追赶的大运卡车，穿越障碍、冲过终点！

---

## 🎮 游戏玩法

### 基本操作

| 按键 | 功能 |
|------|------|
| `↑` / `W` | 加速 |
| `↓` / `S` | 刹车 |
| `←` / `A` | 向左转向 |
| `→` / `D` | 向右转向 |
| `C` | 切换后视镜 |
| `ENTER` / `SPACE` | 开始游戏 / 确认 |

### 游戏目标

在被大运卡车追上之前，冲过终点线！

- 游戏开始 **3 秒后**，大运卡车将全速出发追赶你
- 冲出赛道边界会立即 **Game Over**
- 撞上路障会让你**原地停住**，被大运追上的机会大增
- 安全到达终点线即可**通关**

## 🗺️ 关卡介绍

### 第一关 — 逃离公路

- **赛道类型**：直道 + 弯道组合线路，含急弯、障碍区、护栏区
- **终点**：赛道尽头的红白格终点线
- **大运速度**：`MAX_SPEED × 0.9`（比玩家最高速略慢，但初始距离很近）
- **障碍物**：路中央的木箱、路边护栏收窄路段
- **通关提示**：冲过终点后显示「第一关通关！」，按 `ENTER` 进入第二关

### 第二关 — 圆形竞技场

- **赛道类型**：大型圆形赛道，绕一整圈回到起点
- **终点**：绕圈回到出发点附近的终点线
- **大运速度**：`MAX_SPEED × 1.15`（比玩家最高速**更快**，会持续追近）
- **障碍物**：随机分布的木箱 + 横倒在路上的大树，障碍物偏向路的一侧，始终留有通道
- **通关提示**：成功绕完一圈后显示「成功逃离！YOU WIN!」

## ⚙️ 游戏机制说明

### 伪3D渲染

游戏采用经典**伪3D赛车**设计：物理逻辑在一维坐标系中运行，Three.js 负责将其投影为 3D 画面。弯道通过对玩家施加「离心力偏移」来模拟，玩家需反向打方向盘抵消。

### 速度系统

- 玩家最高速：`MAX_SPEED = SEGMENT_LENGTH × 60`
- 加速响应较快，松开油门会自然减速
- 离路边缘后速度会受惩罚性减速

### 大运 AI

- 出发时间：游戏计时 **3 秒后**固定速度出发，全程匀速
- 第一关速度比玩家慢，但初始距离仅 40 段，碰障碍物或减速就很危险
- 第二关速度超过玩家最高速，持续缩短与玩家的距离

### 后视镜系统

按 `C` 开启后视镜（画面顶部中央），可实时看到大运的追赶距离：
- 大运较远时显示微小车灯光点和「远处追击中」提示
- 大运靠近时车身轮廓变大，并附带危险红色闪烁警告

### HUD 界面

| 元素 | 说明 |
|------|------|
| 左下角速度表 | 实时显示当前速度（km/h），超速变红 |
| 顶部进度条 | 红色=玩家位置，蓝色=大运位置 |
| 右下角距离 | 大运与玩家的段距离，绿→黄→红预警 |

## 🛠️ 技术栈

- **渲染引擎**：[Three.js](https://threejs.org/) v0.169.0（via CDN importmap）
- **语言**：原生 HTML5 + JavaScript（ES Module）
- **无任何构建工具**，直接浏览器打开即可运行

---

## 🚀 本地运行

由于使用了 ES Module importmap，需要通过 HTTP 服务器运行（不能直接双击 html 文件）：

```bash
# Python 3
python3 -m http.server 8080

# 然后访问
http://localhost:8080/index-3d.html
```
